# css100

## 使用方法

1. 克隆本仓库到本地：
    ```bash
    git clone https://github.com/mealcomes/css100.git
    ```
2. 用浏览器打开根目录下的 `index.html`，即可查看我实现的所有 CSS 示例的效果(~~虽然直接点击上面的“效果展示”图标也能看到~~)
